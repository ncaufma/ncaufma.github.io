# Conteúdo Editável — NCA-UFMA

Esta pasta contém os arquivos de conteúdo do site NCA-UFMA.

**IMPORTANTE:** O Astro lê o conteúdo de `src/content/`, não desta pasta.
Esta pasta existe como referência para o administrador.

Ao editar um arquivo aqui, **copie-o também para `src/content/`** ou edite
diretamente em `src/content/` para que as mudanças apareçam no site.

Para desenvolvimento futuro, você pode configurar um script de sincronização
ou simplesmente editar diretamente os arquivos em `src/content/`.
